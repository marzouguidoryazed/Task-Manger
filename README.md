# Task Manager Project
![Node.js](https://img.shields.io/badge/Node.js-v18-green) ![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-blue)
## Description
A full-featured **Task Manager** web application with **Frontend** (HTML, Bootstrap, JavaScript) and **Backend** (Node.js, Express, SQLite). Users can **add, edit, delete, mark as completed, filter, and sort tasks**
## Features
- Add new tasks
- Edit tasks via Bootstrap modal
- Delete tasks
- Toggle task completion
- Filter tasks: All / Completed / Pending
- Tasks sorted by newest first
- Clean and responsive interface with Bootstrap 5
## Tech Stack
- **Frontend:** HTML, CSS, Bootstrap 5, JavaScript (Fetch API)
- **Backend:** Node.js, Express
- **Database:** SQLite
- **Communication:** REST API with JSON
